📰 主要 AI 企業の最新動向

【Microsoft Copilot】機密情報漏洩の重大セキュリティバグ — 8ヶ月間で2回目の信頼境界侵犯

Microsoft Copilot が DLP（Data Loss Prevention）ポリシーと感度ラベルを無視し、4週間にわたって機密メールの読み込みと要約を実行していたことが判明しました。

詳細：

  • 期間： 2026年1月21日から4週間
  • 影響範囲： NHS（英国国民保健サービス）を含む複数の規制環境下の組織
  • 根拠： インシデント ID は NHS INC46740412、Microsoft CW1226324
  • 重大性： これは 8 ヶ月以内に発生した 2 回目の信頼境界侵犯。前回（2025年6月）は「EchoLeak」と呼ばれる、マルウェア侵入や複雑なプロンプトインジェクション攻撃を要さないゼロクリック脆弱性（CVE-2025-32711、CVSS 9.3）だったもの

ビジネス示唆： エンタープライズ AI 導入検討企業は、LLM の「検索拡張生成（RAG）パイプライン」がエンドポイント検出（EDR）や Web アプリケーション・ファイアウォール（WAF）の監視外で動作することを認識する必要があります。従来のセキュリティスタックでは LLM 層での違反を検知できない構造的欠陥があります。

参照： https://venturebeat.com/security/microsoft-copilot-ignoring-sensitivity-labels-dlp-cant-stop-ai-trust-failures

【Meta】AI インフラ拡張戦略 — 再生可能エネルギー 1GW 大量購入

Meta は 1 週間でソーラーパネル 1GW（ギガワット）の導入契約を締結し、3 件の米国内取引により、大規模 AI 訓練・推論に必要な電力基盤を整備しています。

詳細：

  • 容量： 1GW = 100万キロワット（約 50 万世帯の年間消費電力相当）
  • 戦略的背景： LLaMA シリーズの訓練規模拡大、AI推論インフラの低コスト化を実現するため、カーボンニュートラル達成と同時に電力コスト削減を推進
  • 企業の位置づけ： OpenAI・Google・Anthropic に比べてオープンソース路線を強化。エネルギー自給率の向上は、よりアクセスしやすい大規模言語モデル提供を可能にする

ビジネス示唆： AI 企業の競争力は単なる「モデルの性能」から「エネルギー効率 × 推論コスト」へシフト。Meta のコスト優位性戦略は、クラウド業者・SaaS企業向けに安価で高性能な LLM API を提供する道を広げます。

参照： https://techcrunch.com/2025/10/31/meta-bought-1-gw-of-solar-this-week/

📚 arXiv 注目論文の深掘り

【論文1】MolHIT — 離散拡散モデルで分子グラフ生成を革新

要約： MolHIT は、分子グラフの生成に特化した階層的離散拡散モデルです。化学的な妥当性を過去最高水準で実現し、1D（文字列ベース）生成モデルを複数指標で上回りました。

詳細：

  • 手法： 階層的離散拡散（Hierarchical Discrete Diffusion Model）+ 化学知識を組み込んだ原子エンコーディング
  • 成果： MOSES ベンチマークで「グラフ拡散では初めて」ほぼ完全な有効性を達成
  • 応用： 複数特性の誘導生成、スカッフォルド拡張（既知の薬剤骨格を基に新規分子設計）
  • 意義： AI ドラッグディスカバリーにおいて、従来の 1D 文字列モデルより化学的な現実性を保証

ひと言で言うと？ 「化学の法則を守りながら、コンピュータが新しい薬剤候補を自動設計する力が、ついに現実のレベルに達した」

参照： https://arxiv.org/abs/2602.17602

【論文2】AutoNumerics — 物理方程式の数値計算を AI が自動化

要約： AutoNumerics は、自然言語で記述された偏微分方程式（PDE）から、自動的に数値ソルバーを生成・検証するマルチエージェント・フレームワーク。24 個の標準・実用問題で、既存ニューラル解法と同等以上の精度を達成。

詳細：

  • 入力： 自然言語での PDE 説明（「熱拡散方程式を解け」など）
  • 出力： 実装可能な数値ソルバー（古典数値解析に基づいた透明性のあるコード）
  • 検証機構： 残差ベースの自己検証（生成したソルバーが正しいか自動判定）
  • 特徴： ニューラルネットワークのブラックボックス化を回避し、古典的数値解析の理解可能性を保持

ひと言で言うと？ 「複雑な物理シミュレーションに必要な数値計算プログラムを、物理の専門家なしで自動生成できるようになった」

参照： https://arxiv.org/abs/2602.17607

【論文3】HIPE-2026 — 多言語歴史テキストから人物・地名の関係を自動抽出

要約： HIPE（Entities and Relations in Multilingual Historical Texts）-2026 は、ノイズの多い多言語歴史文献から、人物と場所の関係を抽出する評価キャンペーン。知識グラフ構築や歴史的伝記復元など、デジタル人文学への応用を想定。

詳細：

  • タスク： 二種類の関係分類
    • 「at」（その人物は過去にこの場所にいたことがあるか？）
    • 「isAt」（その人物は出版時点でこの場所に位置しているか？）
  • 言語対応： 複数言語・時期にまたがるテキスト
  • 評価軸： 精度（Accuracy）× 計算効率（Efficiency）× 汎化性能（Domain Generalization）
  • 応用： 歴史データの大規模処理、知識グラフの自動構築、空間分析

ひと言で言うと？ 「歴史書から自動的に『いつ誰がどこにいたか』を抽出し、デジタル化された歴史地図を構築する」

参照： https://arxiv.org/abs/2602.17663

📖 用語解説

【DLP】Data Loss Prevention（データ損失防止）

企業システムが機密情報の流出を自動検知・防止する機能。感度ラベル（メールの「社外秘」マーク）と組み合わせ、AI がそのデータにアクセスしないよう制御する。Microsoft Copilot 事例では、この「制御」が RAG パイプライン層で有効に機能しなかった点が問題。

【RAG】Retrieval-Augmented Generation（検索拡張生成）

LLM が外部データベース（過去のメール、ドキュメント、ナレッジベース）を検索してから回答を生成する手法。効果的だが、アクセス権限の管理が困難。

【階層的離散拡散モデル】Hierarchical Discrete Diffusion

従来の拡散モデルを化学的な先験知識で拡張したもの。「原子のタイプ」「化学結合」など化学的なカテゴリを段階的に生成し、物理的に不可能な分子が生成される確率を削減。

【数値ソルバー】Numerical Solver

微分方程式を計算機で近似的に解くプログラム。有限差分法、有限要素法など複数の手法が存在し、PDE の種類に応じた選択が必要。従来は数値解析の専門家が手作業で設計。
